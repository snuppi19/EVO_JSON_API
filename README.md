# EVO_JSON_API_TEST
---
## Mô tả 
Đây là project luyện tập bài tập cơ bản về việc lấy API từ 1 nguồn khác và hiện thị rồi sử dụng vào tác vụ gì đó
## Yêu cầu 
* Java [Phiên bản 17]
* Sử dụng Maven
* IDEA: Intellij
## Tải & Cài đặt 
1. Clone repository
```
https://github.com/snuppi19/EVO_JSON_API.git
```
3. Tải dependencies
 * backend
```
mvn install
```
4. Chạy backend
```
mvn spring-boot:run
```
